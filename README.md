Описание проекта
Проект представляет собой программу, которая получает данные о работодателях и их вакансиях с сайта hh.ru 
с использованием публичного API и библиотеки requests. Полученные данные сохраняются в базу данных PostgresSQL 
с использованием библиотеки psycopg2. Для работы с данными в базе данных создается класс DBManager, 
который предоставляет различные методы для получения информации о компаниях и вакансиях.

- Структура проекта:

I. Папка "database":
1. config.py - параметры подключения к базе данных.
2. employers.py - Класс для получения информации о компаниях и их вакансий из сайта hh.ru.
3. vacancies.py - Класс для работы с базой данных вакансий.
4. manager.py - Класс для управления операциями базы данных с использованием psycopg2.

II. Папка "models":
1. company_vacancy.py - Класс для получения информации о компаниях и их вакансий из сайта hh.ru.

III. Папка "utilities"
1. progress_bar.py - Отображает прогресс выполнения итераций с помощью полосы прогресса.

IV. файл "file_processing.py"
- Функция Получает данные о работодателях и их вакансиях с сайта hh.ru, сохраняет в БД.

V. "файл main.py"
- главный исполняемый файл.


ПРИМЕЧАНИЕ:
В папке database, в файл config.py необходимо внести параметры для подключения к базе данных.
изменить следующие параметры:
1. 'dbname': "" 
2. 'user': ""
3. 'password': ""
4. 'host': ""
